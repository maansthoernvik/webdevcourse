Tillbaka

postion

position kan användas för att styra vart ett HTML-element ska hamna på en sida. Det finns ett antal olika inställingar som kan ges till position:

  • static, som är defaultvärdet.
  • absolute, vilket möjliggör att placera ett element vart som helst på sidan och ignorera platsen där elementet egentligen skulle hamnat. Se inställingarna top, left osv. för information om hur man placerar ett element som fått position satt till absolute.
  • relative, vilket gör att ett element kan placeras relativt till sin nuvarande position i HTML-flödet.
  • fixed, som gör att ett element inte rör på sig, inte ens om webbsidan scrollas. Denna sida har en färgglad fyrkant uppe i vänstra hörnet som är ett exempel på en <div> som har position satt till fixed. Som syns när man scrollar så är denna alltid synlig.

Både absolute, relative och fixed har gemensamt att de alla tre gör det möjligt att använda top, left, osv. Beroende på vilken av de tre som använts så kommer då elementet att positioneras därefter.

Komplement till position

Som nämnt så måste det efter position satts även anges vart elementet ska placeras, annars kommer det inte att synas. För detta kan följande användas:

  • top
  • right
  • left
  • bottom

Alla dessa fyra inställningar tar ett argument, antingen i pixlar, em, % osv. och bestämmer elementets position i förhållande till just den kanten. Exemplet ovan med den fixed- positionerade <div>-taggen har följande: 

        .example_fixed {
          top: 10px;
          left: 10px;
        }

Detta innebär då att den placeras 10 pixlar in från både över och vänsterkant. Man behöver inte ange alla fyra kanter, det räcker vanligtvis med två för att få elementet att hamna där man vill ha det.

z-index

Det är inte bara möjligt att placera element på X och Y-axlarna, det går även att lägga till element på Z-axeln med hjälp av z-index. z-index tar ett nummer som parameter, ju större nummer desto högre upp i "högen" ligger elementet. float

float kan användas för att få till en liknande layout som denna sida har. Inställningen används för att få ett element att "flyta" åt en sida. float accepterar tre argument: right, left och none. Inställningen flyter ett element åt en sida och låter annat material fylla på bakom sig. Denna sida har först använt float: left; på den första kolumnen och sedan float: right; på den högra kolumnen. Mittkolumnen har sedan tilldelats 30% på både margin-left och margin-right då detta är storleken hos de båda sidokolumnerna. Margin behöver justeras då mittkolumnen annars kommer att hamna under den vänstra kolumnen och expandera ut under den högra kolumnen. Margins används alltså för att bestämma storleken hos mittkolumnen. Text kommer inte att hamna under de båda sidokolumnerna, dock kommer behållaren att göra det.

Fotområde med clear

Det här fotområdet hamnar helt fritt här nere i och med att clear har använts. Denna kan sättas till antingen: left, right och both. Denna är användbar om det till exempel finns en massa element på en sida som använder float åt olika håll och man vill att en ny box ska visas på sin egen rad, som denna <footer>. Här har jag använt mig av clear: both; för att vare sig höger eller vänster sida ska få innehålla något element, därmed visas denna box på sin egen rad.
