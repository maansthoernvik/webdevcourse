  • Tabeller skapas med table-taggen. För att strukturera upp sin tabell så kan man först och främst lägga till en överskrift med hjälp av caption-taggen. Caption måste vara det första elementet inuti tabellen för att den ska vara verksam!
    För att lägga till en ny rad till tabellen så använder man tr (för table row), inuti denna så kan man sedan lägga till data och headers (överskrifter) för sina columner. Överskrifter lägger man till mha. th och datan med td. Nedan ser du ett exempel:
    Här är caption
    Kolumn 1 Kolumn 2
    Data 1 Data 2 Data 3
    Som det syns ovan så finns det 3 kolumner data för rad 2, men ingen överskrift för den tredje kolumnen, detta är för att det bara sattes in två th-element i den första raden.
  • Maps kan användas i samband med bilder som beskrevs på sidan 4 för att skapa klickbara områden inuti en bild. Det kan tillexempel vara användbar för kartor för att kunna klicka på olika platser på kartan. Du kan skapa en map genom att använda map-taggen.

    För att skapa ett klickbart område så måste area-taggen användas som ett element inuti map-taggen. Area måste kompletteras med vart inuti bilden som det klickbara området ska placeras. Detta görs genom att använda coords och shape attributen på area-taggen. För en cirkel så spesas först två koordinater, x och y och sedan radien för cirkeln, till exempel: x, y, r. X och Y kommer att vara mittpunkten för den skapade cirkeln. För att lägga till en cirkel så måste då också shape="circle" anges.

    För en rektangel anges istället x1, y1, x2, y2 där 1-paret är det övre vänstra hörnet och 2-paret är det nedre högra hörnet. För att lägga till en rektangel så måste också shape="rect" anges.

    För en månghörning så används shape="poly" och därefter så kan x & y-par läggas till för så många hörn som månghörningen ska ha.

    För att området faktiskt ska vara klickbart så måste även en href attribut läggas till på area-taggen, länka sedan till det som ska hända när området klickats på. Nedan finns ett exempel på en bild med klickbara områden i övre vänstra och nedre högra hörnet.


    För att till sist koppla ihop den skapade mapen med bilden så att bilden förstår att den ska vara klickbar på de spesade ställena, lägg först till ett namn på map-taggen med attributet name. Sätt sedan på img- taggen attributet usemap till exakt det namn som sattes på mapen. Så här:
    <img src="bild.jpg" alt="en bild" usemap="#map">
    <map name="map">
        <area coords="..." shape="...">...
    </map>
