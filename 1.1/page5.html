Listor

Listor finns i ett par olika utformanden som kan väljas mellan beroende på behov.
  • Unordered lists, ul visar bara en indikation per listelement, ingen numrering eller annan indikering som kan särskilja elementen från varandra. Varje listelement sätts in genom li-taggen som måste användas för varje nytt element som ska visas i listan. Den lista som använts för denna sidan är en ul-lista.
  • Ordered lists, ol visar numrering för sina element, som också går att ändra utformning på mha. type attributet på ol-taggen. Type kan sättas till: 1 (default), A, a, I eller i beroende på vilken utformning man vill ha på sin numrering. li-taggen används även här för att skapa nya listelement. Så här kan en ol-lista se ut, denna lista börjar på "A" eftersom att den har attributet type satt till "A":
    1. Första listelementet
    2. Andra listelementet
  • Description lists, dl används för att skapa listor med termer och en beskrivning för varje term. Varje term definieras med dt, beskrivningen ska sedan följa efter, efter sluttaggen för termen, med dt. En beskrivning för en term som definierats med <dt>term</dt> skall direkt följas med en termbeskrivning som läggs till med hjälp av <dd>-taggen.
