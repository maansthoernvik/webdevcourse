Popups

Det finns flera inbyggda popups som man kan använda sig av. alert, prompt och confirm är tre popups som kan användas för lite olika saker. Alert låter användaren veta att någonting skett eller är på väg att ska. Prompt ber användaren om information genom ett text-fält. Confirm används för att be användaren om en bekräftelse.


Öppna nya fönster

En annan form av popups kan tillämpas genom att öppna helt nya fönster. Detta görs genom open- och close-funktionerna.

Den öppnade sidan kan stängas mha. knappen nedan.

Intervall och timeouts

Intervall kan användas för att periodiskt utföra något. Nedan så får vi en div att blinka genom att i ett intervall sätta och ta bort en css-klass som sätter bakgrundfärgen på div-en till limegrön.

Timeouts används som timers och utför något efter en utsatt tid. Knappen nedan sätter igång en timer på 3 sekunder som efter den tiden visar ett rött område.
