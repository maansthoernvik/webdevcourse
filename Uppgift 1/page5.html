Listor

Listor finns i ett par olika utformanden som kan väljas mellan beroende på behov.
  • Unordered lists, ul visar bara en indikation per listelement, ingen numrering eller annan indikering som kan särskilja elementen från varandra. Varje listelement sätts in genom li-taggen som måste användas för varje nytt element som ska visas i listan.
  • Ordered lists, ol visar numrering för sina element, som också går att ändra utformning på mha. type attributet på ol-taggen. Type kan sättas till: 1 (default), A, a, I eller i beroende på vilken utformning man vill ha på sin numrering. li-taggen används även här för att skapa nya listelement.
  • Description lists, dl används för att skapa listor med termer och en beskrivning för varje term. Varje term definieras med dt, beskrivningen ska sedan följa efter, efter sluttaggen för termen, med dt. Så här kan det då se ut:
