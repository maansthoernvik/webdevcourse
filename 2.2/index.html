Fontfamiljer

Fontfamiljer är hur man i CSS väljer vilken font som man vill använda till sin text. Denna kan sättas per element, på en hel klass eller för en tagg, precis som med andra CSS-regler.

Det finns olika typer av fontfamiljer, däribland serif, sans-serif och monospace. Serif-fontar används ofta till långa texter eftersom att fonttypen har små "fötter" på bokstäverna, något som ska göra det lättare och smidigare att läsa. Exempel på serif-fontar är Times New Roman och Times.

Sans-serif-fontar används ofta till överskrifter pga. deras rena och simpla utseende. Arial, Helvetica och Verdana är några exempel.

Monospaced betyder att storleken på varje bokstav är lika stor, detta används till exempel för att visa kodsnuttar i text.

När man lägger till en font-familj på till exempel en paragraf så kan man också spesa backup-fontar. Detta gör man genom att efter den font man helst vill formattera till lägga till fler alternativ, separerat med kommatecken. Sist i denna lista så är det vanligt och bra att lägga till ett generiskt val, såsom bara serif, sans-serif eller monospace. Detta instruerar webbläsaren att välja bland vilken serif, sans-serif eller monospace font som helst som datorn redan installerat, det fungerar alltså som en sorts backupfunktion som man i alla fall får bestämma stil på. 

        body {
            font-family: Arial, Helvetica, sans-serif;
        }

Enheter

xx-small, x-small, small, medium, large, x-large och xx-large kan användas för att ställa in storleken på en font, en bit text. Nedan är syntax för att ställa in fontstorlek. 

        p {
            font-size: xx-small;
        }

Dessa storlekinställningarna har värden som beror på din webbläsares grundstorlek för fontar. Om grundstorleken för en paragraf är 16 pixlar så kommer medium att vara 16 pixlar stor, medans small kommer vara mindre, x-small ännu mindre osv. Motsatt sak händer då för large och uppåt förstås.

% och em-storlekar

Både % och em enheterna liknar small, medium, large osv. % fungerar som så att den taggen som % använts på frågar sin förälder (container) vilket fontstorlek den har och tar sedan %- satsen av den storleken. 100% på font-size sätter alltså font-storleken till densamma som föräldern.

em fungerar faktiskt på precis samma sätt som %, men för att använda em så använder man istället 1 för att representera 100% av förälderns storlek. Se följande exempel: 

        p {
            font-size: 100%;
        }
        p {
            font-size: 1em;
        }

Det koden ovan visar är ett likvärdigt sätt att sätta en fontstorlek på, 1em är 100% och 100% är 1em. Vill man istället har dubbelt så stor text som föräldern använder man 2em eller 200%.

Annat

Ett par andra attribut som är bra att känna till i CSS är color, font och list-style. Color sätter färgen på texten, font är ett uttryck som kan användas för att sätta alla olika typer av text-formattering i ett kommando och list-style används för att, precis som font formatterar text, sätta alla typer av formattering på listor.

font och list-style

Font och list-style har gememsamt att de båda är kortkommandon för att slippa skriva alla olika typer av formattering såsom font-weight, font-style, font-variant osv. för text och list-style-type, list-style-position för listor. Nedan är två olika sätt att formattera text respektive listor både med och utan deras kortkommandon vilket ger den efterföljande texten/listan. 

        p {
            font-style: italic;
            font-weight: normal;
            font-family: Arial, Helvetica, sans-serif;
            font-size: x-large;
        }
        p {
            font: italic normal x-large Arial, Helvetica, sans-serif;
        }

        ul {
            list-style-type: circle;
            list-style-position: inside;
        }
        ul {
            list-style: circle inside;
        }

color

color kan läggas på text för att formattera färgen, så här: 

        p.one {
            color: white;
        }
        p.two {
            color: #00ff00;
        }
        p.three {
            color: rgb(0, 255, 0);
        }
        p.four {
            color: hsl(120, 100%, 50%);
        }
        p.five {
            color: rgba(0, 255, 0, 0.5);
        }

Det finns flera olika sätt att lägga till färg på ett element. Man kan lägga till färg med text, hex- koder, rgb-funktion eller hsl-funktion. Genom att lägga till ett "a" bakom rgb eller hsl så kan man också ställa in transparens, om färgen ska vara genomskinlig. RGB-funktionen tar tre argument, värden för nivån av röd, grön och blå. Det fjärde argumentet för transparens spesas mellan 0 och 1, i exemplet ovan så är alltså paragrafen halvt genomskinlig med sitt värde på 0.5.

hsl(hue, saturation, lightness)

hsl() kan vara lite svårare att vänja sig vid eftersom den har ett något ofamiljärt användningssätt. Det första värdet, hue, väljer färgen. Hue tar ett värde mellan 0 och 360, där 0 och 360 är röd. Låter konstigt men se det som att ha gått runt ett helt varv i en cirkel och att du då är tillbaka på start. Det andra argumentet, saturation, är hur "ren" färgen är i procent från 0 till 100. 0% ger alltid samma gråa färg medans 100% ger än väldigt ren och fin färg som valts i hue. Det sista argumentet (eller näst sista om man vill ställa in transparens) är lightness och bestämmer hur ljus färgen ska vara även här med % mellan 0 och 100. 0 är svart och 100% är en helt vit färg. För att få renast möjliga färg utan att riktigt blanda in ljus så används 50%.
