show, hide och toggle

För att manipulera ett element på sin plats så finns det ett antal alternativ att använda. Show, hide, toggle är tre sätt att manipulera om ett element ska visas eller gömmas. För att göra övergången mellan dessa states så finns det även alternativ att ställa in ett delay för show och hide, spesat i millisekunder, alternativt så kan funktionerna fadeIn eller fadeOut användas. Vill man bara delvis dölja ett objekt så kan man använda sig av fadeTo som sätter transparensen hos ett element till det spesade värdet mellan 0 och 1, där 0 är osynligt och 1 är helt synligt. Toggle-metoderna gör så att om elementet är gömt så visas det och vice versa.

Visningsalternativ

animate

Animationer kan åstadkommas med hjälp av jquerys animate-funktion. Här nere ser vi ett exempel på en animation som också har ett antal länkade sk. callbacks, händelser som utförs då den första animationen gått klart. Detta skapar en kedjeeffekt som visar element efter element. Animationen nedan använder en linjär animationsmodell, denna säkerställer att animationens hastighet hålls bestämd genom hela animationen, till skillnad från swing som animerar olika snabbt beroende på hur långt gången den är. Animationerna nedan är också inställda med olika hastighet, den första ganska långsam men sedan snabbar de på, animation efter animation tills den sista som bara har 150 millisekunder på sig.
