Länkar

Länkar är givetvis en viktig del av i princip alla webbsidor, att designa dem kräver att man vet lite mer kring hur de fungerar och vilka sk. tillstånd länkar kan befinna sig i.

Länktillstånd

En länk som aldrig har klickats på befinner sig i ett "oklickat" tillstånd. Du vet säkert hur det ser ut då länken har sin vanliga, blåa, färg. En länk kan även ha ett "klickat" tillstånd och får då som standard en lila färg. Det finns även två till tillstånd som en länk kan befinna sig i: användarens mus befinner sig på länken och länken håller på att klickas på. Dessa fyra tillstånd har givits fyra pseudo-klasser: :link, :visited, :hover och :active. För att visa hur dessa fyra tillstånd kan designas så har vi här ett exempel som byter ut den gamla vanliga färgkoden. 

      a:link {
          color: orange;
      }
      a:visited {
          color: orangered;
      }
      a:hover {
          color: red;
          background-color: yellow;
          cursor: crosshair;
      }
      a:active {
          color: yellow;
          background-color: red;
      }

Ovan så byter vi ut den blåa färgen mot orange när länken bara ligger där, orangeröd när länken tidigare har klickats på samt röd/gul när vi håller musen över den och till sist vänder vi på färgerna röd/gul när vi klickar på en länk. Se nedan för en representation:
